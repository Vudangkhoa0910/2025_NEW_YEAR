# Lì Xì Năm Mới 2026 🧧

> Website vòng quay lì xì năm mới 2026 cực đẹp với hiệu ứng tuyệt vời!

## 🎯 Tính năng

- 🎰 **Vòng quay may mắn** - Animation mượt mà, hiệu ứng đẹp mắt
- 🧧 **Link dùng 1 lần** - Mỗi link chỉ sử dụng được 1 lần duy nhất
- 💰 **7 mệnh giá lì xì** - 22K, 55K, 66K, 77K, 88K, 99K, 111K
- ✨ **Hiệu ứng đẹp** - Confetti, particles, animations
- 📱 **Responsive** - Hoạt động trên mọi thiết bị
- 🚀 **Tối ưu Netlify** - Sẵn sàng deploy

## 🛠️ Cách sử dụng

### Deploy lên Netlify

1. **Kéo thả folder vào Netlify:**
   - Truy cập [Netlify Drop](https://app.netlify.com/drop)
   - Kéo thả toàn bộ folder `2026` vào

2. **Hoặc sử dụng Git:**
   ```bash
   # Push code lên GitHub
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <your-repo-url>
   git push -u origin main
   ```
   - Kết nối repo với Netlify

### Sử dụng website

1. Truy cập trang chủ
2. Nhập tên người gửi và người nhận
3. Nhập lời chúc (tùy chọn)
4. Click "Tạo Link Lì Xì"
5. Sao chép và gửi link cho bạn bè

### Người nhận

1. Mở link nhận được
2. Xem lời chúc từ người gửi
3. Click "Mở Lì Xì"
4. Quay vòng quay may mắn
5. Nhận lì xì! 🎊
